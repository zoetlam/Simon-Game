Welcome to Simon Game

This game is a memory game where you need to repeat a sequence of colors.

The rules are simple:

  • The game will start with one color.
  • You need to click or tap on the color to repeat the sequence.
  • After each successful round, the game will add one more color to the sequence.
  • If you make a mistake, the game will end.

Let's challenge our brain and have fun playing the Simon Color Game!
